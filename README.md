# Machine-learning_EDA
# Overview

* This project focuses on performing Exploratory Data Analysis (EDA) and building a Machine Learning model to extract insights and make predictions from the dataset. The goal is to         understand data patterns, handle missing values, and improve model performance through preprocessing and feature engineering.

# Tools & Technologies

* Python
* Pandas, NumPy
* Matplotlib, Seaborn
* Scikit-learn

# Project Workflow
* Data Collection
* Data Cleaning & Preprocessing
* Exploratory Data Analysis (EDA)
* Feature Engineering
* Model Building
* Model Evaluation

# Conclusion

This project demonstrates how EDA and preprocessing improve machine learning performance and help generate meaningful business insights.
